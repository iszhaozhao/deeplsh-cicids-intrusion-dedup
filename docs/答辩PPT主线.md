# 毕设答辩 PPT 主线（建议 8 页）

## 第1页：标题页
- 题目、姓名、学号、专业、指导教师
- 学校与学院信息

## 第2页：研究背景与问题
- 入侵检测日志告警风暴问题
- 重复日志带来的三类成本：存储、检索、研判
- 研究目标：高效近似去重 + 可演示系统闭环

## 第3页：总体方案
- 方法链路：预处理 -> 深度编码 -> 哈希检索 -> 判重归并
- 系统链路：前端 -> 后端 -> 算法 -> 数据库
- 技术选型理由（轻量、可复现、可扩展）

## 第4页：系统设计与模块
- 功能模块：登录、导入、参数、任务、结果、统计
- 角色与权限：管理员、运维
- 数据表：`sys_user`、`ids_log`、`dedup_task`、`dedup_result`

## 第5页：关键实现
- Python 训练与评估命令（full 口径）
- 后端任务编排与状态流转
- 结果查询与可视化展示要点

## 第6页：实验设置
- 数据集：CIC-IDS-2017
- 实验规模：`n_flows=12000`, `n_pairs=20000`
- 对比：exact-md5、simhash、baseline-mlp、bigru-deeplsh
- 指标：Accuracy / Precision / Recall / F1 / Compression / Latency

## 第7页：实验结果
- 最优模型：`bigru-deeplsh`
- Accuracy 83.16%，Precision 78.34%，Recall 91.64%，F1 84.47%
- Compression 83.08%，Top-K latency 111.91 ms
- 结论：在可接受时延下获得较高召回与压缩效果

## 第8页：总结与展望
- 已完成：方法+系统+实验闭环
- 局限：真实生产长期验证不足、企业级能力未覆盖
- 展望：更强语义模型、时序上下文、多源实时联动
